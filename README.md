### 구현된 기능
- 메이플스토리M 캐릭터 조회 서비스 기능(MVP)
- 하루동안 가장 많이 조회된 캐릭터 랭킹 순위 구현(MVP)

### 추가 예정인 기능
- 카트라이더 러쉬플러쉬, 바람의나라:연, HIT2, V4 등의 모바일 게임 조회 기능 추가
- 랭킹 페이지를 별도로 제작하여 게임 종류 별, 일일, 주간, 월간 랭킹 구현
- 조회된 캐릭터 비교 기능
- OAuth2를 이용한 회원 가입 기능
- 회원들이 이용할 수 있는 게시판 기능
